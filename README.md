# Flowchart Application

A professional, web-based diagramming application built with React, TypeScript, and React Flow. Designed for creating complex workflows and process diagrams with a premium Miro-like interface.

## 🚀 Features

- **Drag-and-Drop Interface**: Smooth and intuitive workflow creation.
- **Multiple Node Types**: Process, Decision, Start/End, Database, and more.
- **Dynamic Connections**: Intelligent edge routing and conditional styling.
- **Advanced UI/UX**: Zoom/pan, snap-to-grid, alignment guides, and mini-map.
- **State Persistence**: Automatic local state management via Zustand.
- **Rich Design System**: Modern, responsive design with support for multiple themes.

## 🛠️ Technology Stack

- **Frontend**: React 19, TypeScript
- **State Management**: Zustand
- **Diagram Engine**: React Flow
- **Styling**: CSS (Modern standards)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 📦 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 📜 Technical Principles

- Modular, reusable component architecture.
- Clean separation between UI, Logic, and Data.
- High-performance rendering for large diagrams.
- Type-safe implementation throughout the codebase.
